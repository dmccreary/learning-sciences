# Bloom the Elephant — Image Generation Prompts

This page contains seven self-contained AI image prompts for generating Bloom, the pedagogical mascot for *Learning Sciences for Intelligent Textbook Design*. Each prompt is independent — no shared base block to copy. Paste directly into your image tool of choice (ChatGPT/DALL-E, Midjourney, Stable Diffusion, Gemini).

## Character Reference

- **Name:** Bloom
- **Species:** Elephant
- **Catchphrase:** "Let's build a mental model."
- **Personality:** Wise, curious, warm, playful
- **Accessories:** Small round wire-rimmed glasses only (no held objects, no hats, no clothing)
- **Palette:** Soft blue-gray body, cream highlights inside ears, warm blue glasses frames
- **Art style:** Modern flat vector, clean lines, transparent background

## Generation tips

1. Generate at **1024×1024** and downsize for web use.
2. Ensure **fully transparent background** in every pose.
3. After download, trim excess padding with:
   ```sh
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/FILENAME.png
   ```
4. Target display size is 90px inside admonitions — untrimmed padding shrinks the mascot visibly.

## After generation

1. Save each image to `docs/img/mascot/` with the filenames listed above.
2. Run the trim script on each (see "Generation tips" at the top).
3. Preview in-context on the mascot test page at `/learning-graph/mascot-test/`.
4. Regenerate any pose that doesn't match the others — **consistency of face, glasses, and body proportion matters more than any single pose**.

If one AI tool produces inconsistent results across poses, try generating all seven in the **same session** with the **same tool**; cross-tool mixing is the #1 source of identity drift.
