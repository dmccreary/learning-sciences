## About This MicroSim

Experience the chunking effect firsthand. A string of 12 letters is flashed briefly, then you type what you recall. The left panel uses random letters (XQNTRPKVLYBF); the right panel uses letters that form familiar acronyms (FBI IBM CIA NASA). A running tally compares your recall accuracy in each condition.

Use the **Flash duration** slider to control how long letters are visible (500-2000 ms). The **Condition** dropdown selects random only, chunked only, or alternating. Toggle **Show chunk boundaries** to add or remove spaces between chunks.

**Learning Objective (Apply):** Demonstrate that perceived working-memory capacity depends on the chunk structure of the material, not on the raw number of elements.
