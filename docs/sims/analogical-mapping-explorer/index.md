## About This MicroSim

This MicroSim visualizes structure mapping between a source and target domain. The learner drags connections from source elements (left) to target elements (right). Each mapping is scored as a relational match (green), surface-only match (amber), or contradictory (red).

**Controls:**

- **Domain Pair** -- choose from Solar System/Atom or Plumbing/Circuit
- **Show Structure** -- reveals the relational roles on each side (a scaffold that trades speed for retention)
- **Surface Distractors** -- adds surface-only features to increase difficulty
- **Reset** -- clears all mappings

**Learning Objective (Analyze):** Given a source and target domain, identify which elements map on structural grounds, which map only on surface features, and which do not map at all.
