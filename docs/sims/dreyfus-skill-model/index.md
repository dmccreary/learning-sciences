## About This MicroSim

This interactive infographic shows the five Dreyfus stages -- Novice, Advanced Beginner, Competent, Proficient, and Expert -- as a horizontal progression with a rule-following to intuition gradient above.

**Modes:**

- **Explore** -- hover over each stage to see its definition, three behavioral markers, rule-vs-intuition balance, and preferred form of feedback
- **Quiz** -- read a learner description and click the stage that best fits; receive feedback explaining the diagnostic marker

**Learning Objective (Analyze):** Given a description of a learner's performance, place the learner on the Dreyfus skill model and name the transition they are currently approaching.
