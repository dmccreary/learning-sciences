## About This MicroSim

This interactive diagram shows how Learning Sciences draws from four parent disciplines. Click each box to read a description of that discipline in the info panel below the flowchart. A progress bar tracks how many disciplines you have explored, and a celebration animation plays when you have visited all six boxes.

Three research disciplines — Cognitive Science, Educational Psychology, and Neuroscience in Learning — sit on the top row. Instructional Design serves as the translation layer between research and practice. All four converge on Learning Sciences, which in turn feeds into Intelligent Textbook Design.
